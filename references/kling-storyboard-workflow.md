# Kling（可灵）分镜提示词工作流参考

> 基于 2026-06-01 赛博朋克雨夜短片实战记录

## 完整工作流

```
故事稿 → [mmx vision] 角色外观提取 → 委托 @videographer → 第一次交付
→ [用户反馈修正] → 再委托修正 → 最终交付
```

## 关键经验

### 1. 分镜定义：一个分镜=一个连续镜头

用户修正点：「你知道啥叫分镜吗」——每个分镜只能有一个机位、一个场景、一个动作。

| ❌ 错误写法 | ✅ 正确写法 |
|------------|------------|
| 夜市吃东西+别花婚纱亲密瞬间，5s | 分镜A：夜市吃东西，4s；分镜B：别花婚纱，4s |
| 三段回忆快切蒙太奇，6s | 分镜A：海边奔跑，4s；分镜B：夜市，4s；分镜C：别花，4s |

三段回忆必须占3个独立分镜，不能合并。

### 2. 开场设计：眼睛特写优于天台远景

用户修正点：「第一个镜头是从眼睛特写开始，不是从天台开始」

- 情绪化短片开场：极特写眼睛（闭→睁），致敬《银翼杀手2049》
- 避免用远景建立镜头作为开场——会把情绪泄掉
- 开场眼睛特写提示词关键要素：
  - 眼睛从闭合到睁开的慢动作
  - 虹膜纹理清晰可见
  - 瞳孔映射环境元素（霓虹、城市倒影等）
  - 睫毛水珠/眼部妆容细节

### 3. 结尾设计：LED文字逐字闪烁

用户修正点：「文字是一个个闪过，类似二极管灯一样」

- ❌ 错误：眼中闪现文字（整体出现，无动画感）
- ✅ 正确：LED霓虹灯管/点阵屏跑马灯效果，逐字点亮
- 提示词关键描述词：逐字亮起、逐个点燃、跑马灯、点阵屏、霓虹灯管通电闪烁、光晕辉光
- 文字序列：「我」→「爱」→「你」→（熄灭）→「再」→「见」
- 每个字独立点燃，有发光光晕，最后眨眼渐黑

### 4. 角色一致性：锚点嵌入 vs 参考图

Kling 不支持 Seedance 的 `@图片` 引用语法，角色一致性**全靠提示词维持**。

每个分镜提示词首句必须包含完整角色锚点：
```
[发色+发型] + [眼睛颜色+质感] + [脸型+肤色] + [妆容] + [服装]
```

如果故事涉及多套服装切换，每个分镜必须**显式写明当前服装**。

### 5. 回忆转场的雨滴锚点

用「雨滴击中眼睛→闪白过渡」作为现实↔回忆的转场视觉锚点，具体步骤：
1. 极特写眼睛，雨滴慢镜头下落
2. 击中眼球表面瞬间，水滴在虹膜上扩散
3. 闪白，切入/切出回忆
4. 回忆段去掉所有环境音，只留BGM

### 6. 总时长控制

当用户说「只做1分钟以内」时：
- 计算每个分镜的理想时长（Kling每段3-5s）
- 12镜×平均4s=48s，预留12s给剪辑转场，总长≈60s
- 如果故事情感弧线短，10镜即可（10×4s=40s + 转场）
- 格式：在汇总表中列出每个分镜的精确秒数，求和验证

### 7. 双段式提示词结构（最终用户确认格式）

用户修正点：「每个分镜需要分开两部份，一部份是生图使用的，一部份是AI视频使用的」

每个分镜拆为**两段**：

**🎞️ 生图帧描述（50-80字）** — 给 Midjourney/Stable Diffusion 出首帧参考图
- 镜头参数（焦距/光圈/景深/运镜方式）
- 布光描述（主光源位置/色温/光比）
- 构图和环境
- 角色定格式姿态（一个瞬间，没有运动）

**🎥 视频动作描述（80-150字）** — 给可灵 Kling 出视频
- 镜头运动（dolly in/out, rack focus, focus breathing 等）
- 角色动作变化（先A然后B）
- 真人微动作序列（眼皮颤→拇指摩挲→吞咽→手指抖等）
- 情绪/心理在3-5秒内的演变轨迹
- 转场/结束方式

**示例（来自实战）：**

```
## 分镜01 · 眼睛特写开场
⏱ 时长：4秒 | BGM时间戳：0:00–0:04

🎞️ 生图帧描述：
100mm微距极特写，大光圈F1.8浅景深，焦点锁定虹膜纹理——画面完全充满少女的
一只冰蓝色眼睛。主光源来自远处霓虹广告牌的蓝紫色冷光，从侧面45°照亮虹膜纹理，
形成高反差侧光效果，瞳孔暗部几乎全黑。虹膜纹理清晰如琉璃，瞳孔深处隐约映射
远处霓虹灯光的模糊倒影。上下睫毛挂着细密水珠，眼皮半开状态。高反差蓝紫冷色调。

🎥 视频动作描述：
眼睛从闭合状态缓缓睁开，慢动作升格。睁开的瞬间，眼皮末梢有细微的抽搐抖动，
像刚从噩梦中挣扎醒来。眼球表面反射着蓝紫色夜空和雨水的流动光晕，虹膜上的
霓虹倒影随眼球转动微微流动。雨水持续落在睫毛上，每次滴落都让睫毛微微颤动。
背景能听到雨声和远处城市低频嗡鸣。
```

### 8. BGM对齐方法（以Purple Rain为例）

Prince《Purple Rain》5分半钟，按情绪曲线切分为7段对齐分镜：

| 时间 | 歌曲段落 | 对齐分镜 | 剪辑处理 |
|------|---------|---------|---------|
| 0:00-0:04 | 前奏钢琴单音 | 开场眼睛特写 | 雨声+钢琴 |
| 0:04-0:12 | 前奏渐强 | 看手机+打字 | 雨声渐弱 |
| 0:12-0:16 | 主歌低唱 | 仰脸迎雨 | 人声起 |
| 0:16-0:19 | 旋律转折 | 雨滴击眼 | 升调+闪白 |
| 0:19-0:31 | 吉他solo（回忆段） | 三段回忆蒙太奇 | **切掉雨声**，只留吉他 |
| 0:31-0:38 | 副歌"Purple Rain" | 回现实+崩坏 | 鼓点踩雨滴击醒瞬间 |
| 0:38-0:46 | 尾声渐弱 | 黑盒+眼睛+眨眼 | 最后一粒钢琴音配眨眼 |

> 回忆段切掉环境音是点睛之笔——雨声骤停，吉他 solo 干净推出，形成强烈的听觉反差。
